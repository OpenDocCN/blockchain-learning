# 5.1 章节简述

本章节将会分为两个小章节进行分别讲解：

*   前期环境配置
*   IDE 配置及工程目录搭建

## 前期环境配置

该小节主要向大家介绍在实际开发过程中所依赖的一些系统服务组件以及在团队协作办公中所不可或缺的文档管理、代码管理等工具。

## IDE 配置及工程目录搭建

本教程我们采用的是 vscode 工具进行开发的，而本项目因为会涉及多种语言的开发，所以我们需要通过 vscode 安装一些插件以方便提高我们的开发效率； 另外，还会讲述如何在本地快速搭建一个标准化 EOS 工程项目。
